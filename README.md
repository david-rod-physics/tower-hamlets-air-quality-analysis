# Tower Hamlets Air Quality Analysis (2023–2024)

## Project Overview

This project analyses nitrogen dioxide (NO₂) air pollution data collected from diffusion tube monitoring sites in the London Borough of Tower Hamlets during 2023 and 2024.

The objective was to explore differences in air pollution between years, monitoring site types and months using Python, Pandas and Matplotlib.

---

## Objectives

- Clean and prepare real-world environmental datasets.
- Compare annual NO₂ concentrations by monitoring site type.
- Investigate seasonal trends throughout the year.
- Compare air quality between 2023 and 2024.
- Interpret the findings and discuss limitations.

## Main Findings

- Roadside sites recorded the highest NO₂ concentrations.
- Pollution levels were lowest during summer.
- Average NO₂ concentrations decreased from 2023 to 2024.
- Similar seasonal patterns were observed in both years.
